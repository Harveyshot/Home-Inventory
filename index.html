📖 How to Use Home Inventory
🏠 Rooms
Tap a room card to open it and view its items. Use + Add Room to create a new room — choose a name and icon. Tap the ✕ on any room card to delete it and all its items.
📦 Adding Items
Open a room then tap + Item. Fill in the item name (required), plus optional details like value, brand, serial number, and purchase date. These details appear in your insurance report.
📷 Photos, Videos & Receipts
When adding an item, tap Photo Library to pick existing photos or videos, or use Take Photo / Record Video to capture new media directly. Add receipt photos in the Receipts section — they'll be tagged with an orange RECEIPT badge. All images are automatically compressed to save storage.
📋 Insurance Report
Go to File ▾ → Insurance Report. Check or uncheck rooms to include. Tap 🖨 Print / PDF to open a print-ready report. On iPhone: tap the Share icon → Save as PDF. On PC: use the browser print dialog → Save as PDF.
☁️ Syncing Between Devices
To move your data between iPhone and PC:
1. On the source device tap File ▾ → Export Backup
2. Save the file to iCloud Drive, Google Drive, or email it to yourself
3. On the other device tap File ▾ → Import Backup and select the file
Your rooms, items, photos and receipts will all transfer.
🔍 Search
Use the search bar at the top to find items across all rooms by name, brand, serial number, category or notes. Tap a result to jump straight to that item.
💾 Your Data
All data is stored locally on your device using IndexedDB. It is never sent anywhere. Export a backup regularly to avoid losing data if you clear your browser or reinstall the app.
